# Divvy Trips Analysis

## Description
This project involves analyzing Divvy bike-sharing trips for the first quarter of 2019. Using the dataset and Python notebooks, we aim to uncover insights into user behavior, trip durations, and other key metrics related to Divvy's operations.

## Features
- Data analysis of Divvy Trips from Q1 2019.
- Exploratory data analysis (EDA) using Python and Jupyter Notebook.
- Insights into user demographics, trip patterns, and popular stations.
- Visualizations to support findings.

## Dataset
- **Source**: [Divvy Trip Data](https://divvy-tripdata.s3.amazonaws.com/index.html)
- **File**: `Divvy_Trips_2019_Q1.csv`
- **Details**:
  - Contains trip data including start/end times, stations, trip durations, and user types.
  - Data covers the first quarter of 2019.

## Tools and Technologies
- **Programming Language**: Python
- **Libraries**:
  - Pandas
  - NumPy
  - Matplotlib
  - Seaborn
- **Platform**: Jupyter Notebook
